# Datos locales

Este directorio contiene datos y artefactos locales que no deben subirse al repositorio.

- `contracts.db`: base de datos local de contratos.
- `contratos_muestra.json` / `sample_contracts.json`: ejemplos locales.
- `reporte_riesgos_*.html`: reportes generados automáticamente.

## Uso local

Coloca aquí tus archivos de datos o resultados que necesites para trabajar en tu entorno local. No se versionarán ni se subirán a GitHub.

Si quieres compartir ejemplos de datos de forma segura, crea un archivo separado con datos anonimizados fuera de este directorio.
